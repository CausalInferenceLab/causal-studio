# Causal Studio

**🌐 언어:** [← English](/) | **한국어**

인과추론 교육 영상과 실행 가능한 코드북을 제공하는 플랫폼입니다.

YouTube에서도 영상 시리즈를 볼 수 있습니다: <https://www.youtube.com/@CausalStudio>

## 학습 방법

1. **영상 시청**: 각 챕터의 명확한 애니메이션으로 개념을 시각적으로 이해합니다.
2. **노트북 실행과 탐구**: Binder 또는 Colab에서 직접 코드를 실행하며 실습하고, 노트북의 상세 코드와 설명으로 더 깊이 이해합니다.
