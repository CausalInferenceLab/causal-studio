# Causal Studio

**🌐 Language:** **English** | [한국어 →](/intro-ko)

A platform providing educational videos and executable notebooks on causal inference. 

You can also find the video series on YouTube: <https://www.youtube.com/@CausalStudio>

## How to Learn

1. **Watch the videos**: Understand concepts visually through clear animations for each chapter
2. **Run and explore the notebooks**: Practice hands-on by running code directly in Binder or Colab, then build a deeper understanding with the detailed code and explanations in the notebooks
